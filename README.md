Una aplicación de consola desarrollada en **C#** y **.NET** que permite realizar operaciones matemáticas básicas. Este proyecto forma parte de mi ruta de aprendizaje en .NET y está enfocado en comprender los fundamentos del lenguaje y la estructura de una aplicación de consola.

---

# 📖 Descripción

La Calculadora permite al usuario ingresar dos números y seleccionar una operación matemática desde un menú interactivo.

Actualmente soporta las siguientes operaciones:

- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División
- 🟣 Potencia
- 🟟 Módulo (Residuo)

---

- ✅ Solicitar dos números al usuario.
- ✅ Mostrar un menú de operaciones.

- ✅ Validar la división entre cero.
